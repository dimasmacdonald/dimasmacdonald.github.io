Purrfect Pet Sitters

As part of a web development assignment, I had to create a home page for a hypothetical pet sitting website. Its name was Purrfect Pet Sitters, which I thought was a pretty funny, original pun until I did some research.

The goals of this assignment focused on creating a responsive web page that effectively used the Bootstrap 4 (v4.0.0-alpha.5) framework and Backstretch, Hover.css, and Blue Imp Gallery plug-ins.

All photos used in the project came from the very talented Unsplash users Carol Hu, @kitsanoo, Lucia Velez, Jill Heyer, Shubhankar Sharma, and Javid Bunyadzade.
