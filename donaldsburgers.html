Donald's Burgers

As part of a web development assignment, I had to create two web pages for a fictional restaurant website. Donald's Burgers is entirely my creation, and we proudly serve the Lorem Ipsum burger fresh off the grill.

Creating Donald's Burgers was both a technical and design assignment. The web pages tested how well I could design elements adhering to the box model and Flexbox layout model.

All photos used in the project came from the very talented Unsplash users Quentin Lagache and Michael Browning.
